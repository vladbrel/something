Функциональные возможности магазина:
1. Создание моделей каталога продуктов, добавление их на сайт администрирования
и создание осн. представлений для отображения каталога.
2. Создание системы корзин для покупок с помощи сессий(cookie django),
для того чтобы пользователи могли сохранять выбранные продукты.
3. Создание форм и функциональных возможностей для соверш. заказа
4. Отправка по эл. почте подтверждения пользователем размещения заказа
